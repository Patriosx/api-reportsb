# RSB (report stolen bikes) API

Backend Coding Challenge.

Application for users to report their stolen bikes

Teck Stack used:

<ul>
<li>NodeJS</li>
<li>Express</li>
<li>MongoDB</li>
<li>Mongoose</li>
</ul>

As user you can:

<ol>
<li>Register</li>
<li>Login</li>
<li>Report stolen bike</li>
</ol>

## Installation

1. Clone the repo

```bash
https://github.com/Patriosx/api-reportsb.git
```

2. Use the package manager [npm](https://www.npmjs.com/package/npm) to install RSB API.

```bash
npm install
```
